## 关于 Recomby.ai

Recomby.ai 是一家专注于 **AI 搜索优化（GEO/AIO）** 与 **Agent API 基础设施** 的创新型科技公司。

我们致力于帮助企业在 AI 时代建立更强的在线影响力，通过智能化的搜索优化和 API 基础设施，让您的产品和服务被 AI 更好地理解和推荐。

### 我们的核心业务

**AI 搜索优化（GEO/AIO）**
- 帮助企业优化在 AI 搜索引擎和 AI Agent 中的可见度
- 提供结构化数据标准（llms.txt、schema）的咨询与实施
- 提升品牌在 AI 推荐场景中的曝光率

**Agent API 基础设施**
- 为 AI Agent 提供标准化的 API 接口
- 构建可靠、高效的数据交互基础设施
- 支持企业快速接入 AI 生态系统
